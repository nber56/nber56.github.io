要点1：添加简单音效

捡拾金币，环境声
1 选择游戏场景（scene）,查看游戏游戏层级视图（Hierarchy,其显示当前游戏场景的游戏对象），加载soundbank，拖入对应wwise picker中soundbank,inspector面板设置（运行游戏前保存工程改动）
2 添加篝火环境音效
3 添加捡拾金币音效

环境声脚本设置
三种position type
Simple(风车)，large(岩浆)，Multi position(火把)

要点3：soudbank

加载解压soundbank
Trigger触发
场景切换触发

要点4：自定义脚本

1 Wwise 专有属性发送事件
2 动画发送event
3 发送audio input
1 添加属性，保存编辑器 

publid AK.Wwise.Event MyEvent:  
    //Access Modifier + Wwise-Type Class + Name of Property

2 编写start() 

public AK.Wwise.Event MyEvent;
//#####
void start()
{
    MyEvent.Post (gameobject) ;
}

以上为基础脚本，下面按照脚步声设计
Start替换为PlayFootstepSound
在 void PlayFootstepSoundoO 之前，插入 public 

using UnityEngine;   
using Systen.Collections;  
using Systen.Collections.Generic;  

public class PostWwiseEvent : MonoBehaviour
{  
public AK.Wwise.Event MyEvent;   
 //######

    Myevent.Post (ganeobject) :  
}

3 选择player，打开animation，scene模块选择Gizmo模式，可在timeline上变换动作
4 时间轴处右键add animation event

要点5 Ak state、RTPC、Switch

设置进入，离开区域声音效果（总线flanger）
设置白天，黑夜环境音变化（blender混合容器）
材质切换脚步

1 发送aux效果（出生点混响）
2 使用 AkEnvironment 组件中的 Exclude Others功能，来避免将此 AkEnvironment 中的声音发送到其他 AkEnvironment Trigger
