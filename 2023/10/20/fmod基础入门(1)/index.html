教程卡丁车

FMOD - Integration Tutorial_Karting Microgame

  1. 使用模板Karting Microgame(unity2020.3.48f1c1)和example fspro(fmod2.02.17)
  2. 在fmod中build
  3. 在unity中导入fmod for unity
  4. unity设置
    1. 选择fmod工程
    2. Listener，audioengine，改为fmod
    3. 其他多项设置
  5. 添加声音：卡丁车引擎
    挂接 添加event emitter
    设置参数1000为默认rpm
    脚本关联 在自定义程序集上添加fmodunity程序集，apply一下

修改脚本 

using UnityEngine;
namespace KartGame.KartSystems
{
    /// <summary>
    /// This class produces audio for various states of the vehicle's movement.
    /// </summary>
    public class ArcadeEngineAudio : MonoBehaviour
    {
        public float minRPM = 0;
        public float maxRPM = 5000;
        ArcadeKart arcadeKart;
        void Awake()
        {
            arcadeKart = GetComponentInParent<ArcadeKart>();
        }
        void Update()
        {
            float kartSpeed     = arcadeKart != null ? arcadeKart.LocalSpeed() : 0;
            // set RPM value for the FMOD event
            float effectiveRPM = Mathf.Lerp(minRPM, maxRPM, kartSpeed);
            var emitter = GetComponent<FMODUnity.StudioEventEmitter>();
            emitter.SetParameter("RPM", effectiveRPM);
        }
    }
}

  1. 添加声音：氛围

  2. 添加声音：音乐
    fmod中创建事件Kart Music
    导入音乐，创建循环
    为事件分配混音
    挂接
    联调
    混音器界面底部打卡，再build

  3. 注意：在开始添加 FMOD Studio 事件之前，最好删除项目中当前存在的预先存在的 Unity 音频源。
    否则出现许多报错
